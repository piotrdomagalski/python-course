Tuple packing and unpacking


>>> x = (1, 2)
>>> a, b = x
>>> a
1
>>> b
2
>>> a, b = b, a
>>> a
2
>>> b
2

Whitespace


if x < 0:
    print('Negative')
elif x == 0:
    print('Zero')
elif x == 1:
    print('Single')
else:
    while x > 0:
        print('More')
        x -= 1

Always indent with 4 spaces!

Style

  • lower_case for functions, methods, variables
  • lower_case or ALL_CAPS for constants
  • CamelCase for classes
  • Attributes: interface, _internal

See PEP8 for more info

Lambda


>>> n = 1, 2, 3, 4, 5
>>> def adder(x, y): return x + y
>>> sum(n) == reduce(adder, n)
True
>>> sum(n) == reduce(lambda x, y: x + y, n)
True

mymodule.py


MYCONST = 3.14
_some_stuff = 'Do not touch this!'

def somefunc(x):
    return x * 2

print('This will always be printed!')

if __name__ == '__main__':
    print('Only printed when run {}'.format(somefunc(3)))

In REPL


>>> import mymodule
This will always be printed!
>>> mymodule.somefunc(3)
9
>>> mymodule.MYCONST
3.14
>>> mymodule._some_stuff
'Do not touch this!'
>>> reload(mymodule)
This will always be printed!

List comprehensions


>>> [x**2 for x in range(10)]
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

>>> [x**2 for x in range(10) if x % 2 == 0]
[0, 4, 16, 36, 64]

>>> [(x, x**2) for x in range(6)]
[(0, 0), (1, 1), (2, 4), (3, 9), (4, 16), (5, 25)]

>>> freshfruit = ['  banana', '  loganberry ', 'passion fruit  ']
>>> [fruit.strip() for fruit in freshfruit]
['banana', 'loganberry', 'passion fruit']

>>> from math import pi
>>> [str(round(pi, i)) for i in range(1, 6)]
['3.1', '3.14', '3.142', '3.1416', '3.14159']
